# User Authentication System

## 📌 Project Overview

This project was developed as part of the ApexPlanet Web Development Internship.

It is a complete User Authentication System developed using PHP and MySQL with secure login, user management, responsive design, profile management, and API integration.

---

## 🚀 Features

### Task 1
- User Registration
- User Login
- Password Hashing
- Session Management
- Logout

### Task 2
- CRUD Operations
- Add User
- Edit User
- Delete User
- View Users

### Task 3
- Form Validation
- Improved User Interface
- Session Security
- Error Handling

### Task 4
- Responsive Design
- Modern CSS Styling
- Mobile-Friendly Layout
- Improved Dashboard
- Responsive User Management Table

### Task 5
- User Roles (Admin/User)
- Admin Access Control
- Profile Management
- Profile Picture Upload
- Currency Exchange API Integration

## ⚙ Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project into `htdocs`.
4. Import the MySQL database.
5. Run:

```
http://localhost/user_auth
```
